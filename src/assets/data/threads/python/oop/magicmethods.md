## Magic (Dunder) Methods
---

Dunder (double-underscore) methods let your objects integrate seamlessly with Python's built-in syntax and operators. They are called automatically by Python in response to specific operations — you never call them directly.

---

## 1. Object Representation

```python
class Vector:
    def __init__(self, x: float, y: float):
        self.x = x
        self.y = y

    def __repr__(self) -> str:
        """Unambiguous developer representation. Used by repr() and in the REPL."""
        return f"Vector({self.x!r}, {self.y!r})"

    def __str__(self) -> str:
        """Human-readable. Used by print() and str(). Falls back to __repr__ if not defined."""
        return f"({self.x}, {self.y})"

    def __format__(self, spec: str) -> str:
        """Supports f-string format specs: f'{v:.2f}'"""
        return f"({self.x:{spec}}, {self.y:{spec}})"
```

## 2. Arithmetic Operators

```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __add__(self, other: "Vector") -> "Vector":
        return Vector(self.x + other.x, self.y + other.y)

    def __sub__(self, other: "Vector") -> "Vector":
        return Vector(self.x - other.x, self.y - other.y)

    def __mul__(self, scalar: float) -> "Vector":
        return Vector(self.x * scalar, self.y * scalar)

    def __rmul__(self, scalar: float) -> "Vector":
        """Called when the left operand doesn't support the operation: 3 * v"""
        return self.__mul__(scalar)

    def __neg__(self) -> "Vector":
        return Vector(-self.x, -self.y)

    def __abs__(self) -> float:
        return (self.x**2 + self.y**2) ** 0.5

v1 = Vector(1, 2)
v2 = Vector(3, 4)
v1 + v2     # Vector(4, 6)
3 * v1      # Vector(3, 6)
abs(v1)     # 2.236...
```

## 3. Comparison Operators

```python
from functools import total_ordering

@total_ordering   # generates missing comparison methods from __eq__ and __lt__
class Money:
    def __init__(self, amount: float, currency: str = "USD"):
        self.amount = amount
        self.currency = currency

    def __eq__(self, other: object) -> bool:
        if not isinstance(other, Money):
            return NotImplemented
        return self.amount == other.amount and self.currency == other.currency

    def __lt__(self, other: "Money") -> bool:
        if self.currency != other.currency:
            raise ValueError("Cannot compare different currencies")
        return self.amount < other.amount

    def __hash__(self) -> int:
        return hash((self.amount, self.currency))

m1 = Money(10)
m2 = Money(20)
m1 < m2    # True
m2 > m1    # True  — generated by @total_ordering
```

## 4. Container Protocol

```python
class Deck:
    def __init__(self):
        self._cards = [f"{v}{s}" for s in "♠♥♦♣" for v in "A23456789TJQK"]

    def __len__(self) -> int:
        return len(self._cards)

    def __getitem__(self, index):
        return self._cards[index]

    def __setitem__(self, index, value):
        self._cards[index] = value

    def __contains__(self, card) -> bool:
        return card in self._cards

    def __iter__(self):
        return iter(self._cards)

deck = Deck()
len(deck)        # 52
deck[0]          # "A♠"
"A♠" in deck     # True
for card in deck: ...
deck[0:5]        # first 5 cards — slicing works via __getitem__
```

## 5. Context Manager Protocol

```python
class ManagedFile:
    def __init__(self, path: str, mode: str = "r"):
        self.path = path
        self.mode = mode
        self.file = None

    def __enter__(self):
        self.file = open(self.path, self.mode)
        return self.file

    def __exit__(self, exc_type, exc_val, exc_tb):
        if self.file:
            self.file.close()
        # Return False (or None) to propagate exceptions
        # Return True to suppress exceptions
        return False

with ManagedFile("data.txt") as f:
    content = f.read()
```

## 6. Callable Objects

```python
class Multiplier:
    def __init__(self, factor: float):
        self.factor = factor

    def __call__(self, value: float) -> float:
        return value * self.factor

double = Multiplier(2)
double(5)     # 10
double(3.14)  # 6.28
callable(double)  # True
```

## 7. Attribute Access

```python
class LazyLoader:
    def __init__(self):
        self._cache = {}

    def __getattr__(self, name: str):
        """Called ONLY when normal attribute lookup fails."""
        if name.startswith("_"):
            raise AttributeError(name)
        self._cache[name] = f"loaded:{name}"
        return self._cache[name]

    def __setattr__(self, name: str, value):
        """Called on EVERY attribute assignment."""
        super().__setattr__(name, value)   # use super() to avoid recursion!

    def __delattr__(self, name: str):
        super().__delattr__(name)
```

> `__getattr__` is called only when normal lookup fails.  
> `__getattribute__` is called on **every** attribute access — override with extreme care.

## 8. Key Dunders Reference

| Dunder | Triggered by |
|--------|-------------|
| `__init__` | Object construction |
| `__repr__` | `repr()`, REPL display |
| `__str__` | `print()`, `str()` |
| `__len__` | `len()` |
| `__getitem__` | `obj[key]` |
| `__setitem__` | `obj[key] = val` |
| `__contains__` | `val in obj` |
| `__iter__` | `for x in obj` |
| `__next__` | `next(obj)` |
| `__enter__` / `__exit__` | `with obj:` |
| `__call__` | `obj()` |
| `__eq__` | `==` |
| `__lt__` | `<` |
| `__hash__` | `hash()`, dict keys, set membership |
| `__add__` | `+` |
| `__bool__` | `bool()`, truthiness |
| `__del__` | Object destruction (finaliser) |
